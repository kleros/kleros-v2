## Contributing

### Pre-Requisites

If you haven't already, you need to follow all the previous steps of the **Contributing** section of the repo's [Contribution Guidelines](../CONTRIBUTING.md)

### Getting Started

#### Compile the contracts

```bash
$ cd kleros-v2/contracts
$ yarn build
```

#### Start the frontend

```bash
$ cd kleros-v2/web
$ yarn start
```
